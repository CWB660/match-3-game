## 🎮 游戏玩法

1. **点击消除** - 点击相同的方块，组成3个消除它们
2. **避免槽位占满** - 如果槽位占满，则游戏结束

## 🚀 运行方法

1. 下载或克隆项目文件
2. 在浏览器中打开 `index.html` 文件
3. 开始游戏！
